A simple console-based Bank Account Management System built using Object-Oriented Programming (OOP) in C++.
This project allows users to create and manage bank accounts with basic operations such as deposit, withdrawal, search, and delete.

Features:
Create a new bank account
Deposit money
Withdraw money
Check account balance
Search for an account by account number
Display all accounts
Delete an account

Concepts Used:
Classes & Objects
Constructors
Vectors
Encapsulation
Modular Code (separate .h and .cpp files)
